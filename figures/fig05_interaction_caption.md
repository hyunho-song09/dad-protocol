# Figure 5 Caption — Nature Protocols Format

**Figure 5 | Schematic illustration of DAD interaction profiling output (Stages 9–11).**
This figure is a matplotlib schematic and does not show actual atomic coordinates or a data-derived structural pose. It illustrates the types of output produced by DAD Stages 9–11 for a representative protein–metabolite pair: a receptor represented as a rainbow N→C cartoon (blue = N-terminus, red = C-terminus), a dipeptide ligand shown as green sticks, a transparent pocket surface derived from P2Rank top-1 pocket coordinates, and labelled contact residues within a 5 Å radius of the ligand. Binding scores shown (Vina affinity = −5.61 kcal/mol, CNN pose score = 0.900, CNN affinity = 4.416) are the Tier 1 ground-truth values from the *F. islandicum* AW-1 primary-study replay for the MCP–Ala-Ile pair; they are reproduced exactly by the DAD pipeline (Δ = 0.000 for all three metrics; see validation_table.tsv). A companion PyMOL script (`fig05_interaction_pymol.pml`) in `figures/source_code/` is provided as a template for generating true three-dimensional structural renders from any archived receptor PDB and docked SDF file when PyMOL ≥ 2.5 is available; the script currently demonstrates the workflow using the RCSB co-crystal example `TAR_ASP_2LIG` (an example PDB only, not a Tier 1 result).
