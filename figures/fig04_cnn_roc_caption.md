**Figure 4 | GNINA CNN pose score as a cross-case classifier for top-pose success in the 16-case RCSB seed.**
**(a)** Scatter plot of CNN pose score versus top-pose RMSD to crystal pose for 16 RCSB seed redocking cases. Each point represents one case; PASS cases (top-pose RMSD ≤ 2.0 Å, green circles) and FAIL cases (top-pose RMSD > 2.0 Å, vermillion triangles) are distinguished. The dashed regression line illustrates the negative cross-case correlation (Pearson r = −0.889, p < 0.001): across cases, a higher CNN pose score is associated with lower top-pose RMSD. The dotted red line marks the 2.0 Å threshold. **(b)** Receiver-operating-characteristic (ROC) curve for binary top-pose PASS/FAIL classification using CNN pose score as the predictor (n = 16 cases). In this 16-case seed, CNN pose score is a strong cross-case classifier for top-pose PASS/FAIL (AUROC = 0.958, AUPRC = 0.987, EF@25% = 1.333). However, it does not eliminate within-case ranking failures: 4/16 cases fail the top-pose criterion and 7/16 cases have a better-RMSD pose at a non-top index (see Supplementary Figure S1). The diagonal dashed line represents a random classifier (AUROC = 0.5). All metrics (Pearson r, AUROC, AUPRC, EF@25%) are computed directly from the source TSV at figure generation time; no values are hard-coded. Best-of-9 RMSD shows weaker cross-case correlation with CNN score (Pearson r = −0.388, p = 0.14), consistent with the diagnostic role of multi-mode sampling rather than prospective pose selection. Source data: `Publication/figures/source_data/fig04_source_data.csv`; computed metrics: `Publication/figures/source_data/fig04_metrics.csv`.
