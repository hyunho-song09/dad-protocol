# Figure 2 Caption — Nature Protocols Format

**Figure 2 | Pre-docking biological triage decision tree (DAD Stage 3).**
Sequential five-rule filter applied to each ORF translation before AlphaFold structure prediction. **R1 (length)**: sequences shorter than 50 amino acids are excluded because AlphaFold2 pLDDT estimates are unreliable below this threshold (Jumper et al., *Nature* 2021). **R2 (signal peptide)**: DeepTMHMM (Hallgren et al., *bioRxiv* 2022) or Phobius (Käll et al., *J. Mol. Biol.* 2004) cleavage-site annotation routes secreted proteins to mature-form docking (PASS_CLIPPED). **R3 (transmembrane helix count, nTM)**: soluble proteins (nTM = 0) proceed intact; single- and double-pass receptors (nTM 1–2, e.g. bacterial MCPs) yield their extracellular or periplasmic domains; multi-pass proteins (nTM 3–6) are clipped to the longest non-TM loop if ≥ 60 amino acids; polytopic proteins (nTM ≥ 7) are excluded. **R4 (dock-eligible region length)**: trimmed sequences shorter than 50 amino acids are excluded because P2Rank cannot reliably predict pockets in fragments of this size. **R5 (functional class)**: HMMER/Pfam annotation boosts priority for canonical metabolite-sensor families (MCP, CRP/FNR, SBP) and downranks hypothetical/DUF proteins. Shaded boxes show the triage outcome for the three *F. islandicum* AW-1 Tier 1 receptor examples: NA23_RS01195 (MCP, nTM = 2, PASS_CLIPPED), NA23_RS08105 (CRP/FNR, nTM = 0, PASS), NA23_RS00870 (RbsB, SP+, PASS_CLIPPED). No existing multi-target docking pipeline provides an automated pre-docking biological triage layer equivalent to Stage 3.
